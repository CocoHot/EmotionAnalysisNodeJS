# NodeJS test for Emotion Analysis #

This simple NodeJS + Sockets.io application is a simple adthat is used to test
[EmotionAnalysis][1] twitter-kafka-spark streaming application.

## How to run ##

Run 
```sh 
npm install 
node src/index.js
```
